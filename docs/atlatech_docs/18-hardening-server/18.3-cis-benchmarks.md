# 18.3 CIS Benchmarks Ubuntu Server

## Application des recommandations CIS

**Installation de l'outil d'audit :**

```bash
sudo apt install lynis -y
```

**Exécution de l'audit :**

```bash
sudo lynis audit system
```

**Résultat attendu :**

```
Hardening index : 78/100
```

**Corrections principales appliquées :**

### 1. Désactivation des services inutiles

```bash
sudo systemctl disable bluetooth.service
sudo systemctl disable cups.service
sudo systemctl mask avahi-daemon.service
```

### 2. Configuration SSH durcie

```bash
sudo nano /etc/ssh/sshd_config
```

```
PermitRootLogin no
PasswordAuthentication no
PubkeyAuthentication yes
MaxAuthTries 3
Protocol 2
```

### 3. Audit du système

```bash
sudo systemctl enable auditd
sudo systemctl start auditd
```

**Audit CIS complet disponible dans :** `/var/log/lynis.log`
