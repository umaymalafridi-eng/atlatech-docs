# 18.4 SELinux/AppArmor

## Configuration AppArmor pour AtlasTech

**Vérification du statut :**

```bash
sudo aa-status
```

**Profils actifs pour Apache :**

```bash
sudo aa-enforce /etc/apparmor.d/usr.sbin.apache2
```

**Création d'un profil personnalisé pour AtlasTech :**

```bash
sudo nano /etc/apparmor.d/atlastech.profile
```

**Contenu :**

```
#include <tunables/global>

/var/www/html/atlastech/** {
  #include <abstractions/base>
  #include <abstractions/apache2-common>

  /var/www/html/atlastech/** r,
  /var/www/html/atlastech/uploads/** rw,
  /tmp/** rw,
  
  deny /etc/shadow r,
  deny /etc/passwd w,
  deny /root/** rw,
}
```

**Activation :**

```bash
sudo apparmor_parser -r /etc/apparmor.d/atlastech.profile
sudo aa-enforce atlastech.profile
```

**Vérification :**

```bash
sudo aa-status | grep atlastech
```
