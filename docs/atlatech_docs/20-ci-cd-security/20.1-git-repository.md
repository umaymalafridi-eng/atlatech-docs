# 20.1 Git Repository

## Repository GitHub AtlasTech

**Déjà configuré selon resume.pdf :**

```bash
cd /var/www/html/atlastech
sudo git init
sudo git remote add origin https://github.com/umaymalafridi-eng/atlastech.git
```

## Commits sécurisés

**Hook pre-commit pour détecter secrets :**

```bash
nano .git/hooks/pre-commit
```

```bash
#!/bin/bash
if git diff --cached | grep -E 'password|secret|api_key' ; then
    echo "ERREUR: Possible secret détecté !"
    exit 1
fi
```
