# 19. Monitoring & Honeypot

## Vue d'ensemble

Surveillance en temps réel des accès au site web, journalisation des modifications CRUD, et déploiement de honeypots pour détecter les attaques.

## Sections

### 19.1 Surveillance Accès Site Web
Logs Apache, détection d'anomalies, dashboards.

### 19.2 Logs CRUD (Modifications Employés)
Traçabilité complète : qui a modifié quel employé et quand.

### 19.3 Wazuh SIEM
Centralisation des logs et détection d'intrusions.

### 19.4 Cowrie Honeypot
Piège SSH pour capturer les tentatives d'intrusion.
